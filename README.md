# Texas Hold'em Poker Multiplayer Server App
Open-source Texas Hold'em poker multiplayer game / 開源德州撲克多人遊戲 / 开源德州扑克多人游戏, designed for real-time gameplay, room-based matches, and easy local deployment.

## Highlights / 亮点 / 亮點
- Real-time multiplayer poker experience / 实时多人扑克牌局体验 / 即時多人撲克牌局體驗.
- Room-based match flow / 基于房间的对局流程 / 基於房間的對局流程.
- Clean game logic and server structure / 清晰的游戏逻辑与服务端结构 / 清晰的遊戲邏輯與服務端結構.
- Easy to run, test, and extend / 易于运行、测试和扩展 / 易於運行、測試與擴展.

## Features / 功能 / 功能
- Create or join a room / 创建或加入房间 / 建立或加入房間.
- Real-time card dealing and betting rounds / 实时发牌与下注回合 / 即時發牌與下注回合.
- Game state sync for all players / 所有玩家的游戏状态同步 / 所有玩家的遊戲狀態同步.
- Hand evaluation, pot settlement, and round management / 牌型判断、底池结算与回合管理 / 牌型判斷、底池結算與回合管理.
- Suitable for learning, customization, and further development / 适合学习、定制和二次开发 / 適合學習、客製化與二次開發.

## Quick Start / 快速开始 / 快速開始
1. Clone the repository / 克隆仓库 / 複製倉庫.
2. Install dependencies / 安装依赖 / 安裝依賴.
3. Run the server and client / 运行服务端和客户端 / 執行服務端與客戶端.
4. Open the game in your browser or local environment / 在浏览器或本地环境中打开游戏 / 在瀏覽器或本地環境中開啟遊戲.
